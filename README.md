# C++ Socket RPG (ターミナル通信対戦)

Linuxの低レイヤーなネットワーク通信（ソケットプログラミング）をゼロからC++で構築し、TCP/IPの基礎論理を学ぶために作成したターミナル型のチャットRPGシステムです。

## 概要
既存のフレームワークに頼らず、OSレベルのシステムコール（`socket`, `bind`, `listen`, `accept`, `connect`）を直接叩くことで、双方向のストリーミング通信を実装しています。

- **サーバー側（魔王）:** ポートを開放し、クライアントからの接続を待機。
- **クライアント側（勇者）:** IPアドレスとポートを指定してサーバーへ突撃。

## 使用技術
- **言語:** C++
- **環境:** Linux (WSL2 / Ubuntu)
- **ネットワーク:** TCP/IP (Socket API)
- **コア概念:** ポインタ操作、無限ループによる通信維持、ファイルディスクリプタ
